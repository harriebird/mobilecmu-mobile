In order to use this App, you must first login using your FES (Faculty Evaluation System) account.
The button on the top left of the screen will toggle the main menu which helps you in navigating the app. You can also access the main menu by doing a swipe gesture on the screen from left to right.
The button on the top right of the screen will toggle the popover menu which contains the buttons linking to Settings, About and Logout.
The Student Information displays the basic profile of a student.
The Grade Report Card displays grades gained in every semester. It uses color coding depending on the grade gained on each subject.

Grade Color Coding:

1 - 2
>2 - 3
5
INC
The Registration Details displays same information found on a Certificate of Registration like subject load and assessment fees.
The Subsidiary Ledger displays ledger transactions of a student.

Ledger Color Coding:

Debit
Credit
This INC Monitor displays all the subjects that gained a grade of INC. It has a countdown timer for the student to comply the subject and uses same color coding with the Grade Report Card.
This app uses pull-to-refresh in order to reload most of its contents. To use this feature, swipe the screen from top to bottom.
Inside settings, you can find the API URL of the app. Do not try to override this unless instructed. Overriding the API URL incorrectly may cause the app not to connect with the server.
